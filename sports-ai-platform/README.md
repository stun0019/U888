# Sports AI Platform

A multi-sport data aggregation platform providing live scores, schedules, and match results.

---

## Features

- Football matches (API-Football)
- NBA games (API-Basketball)
- MLB games (MLB Stats API)
- Match filtering system
- Date selection (UTC+8)
- Live score display
- Finished match results

---
